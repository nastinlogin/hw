# Программа Бухгалтерия

Простая программа для демонстрации структуры модулей и пакетов в Python.

## Структура проекта
- `main.py` - главный модуль для запуска
- `application/salary.py` - модуль с функцией расчета зарплаты
- `application/db/people.py` - модуль с функцией получения сотрудников

## Используемые пакеты
- `python-dateutil` - расширенные возможности работы с датами
- `faker` - генерация тестовых данных

## Запуск
python main.py
